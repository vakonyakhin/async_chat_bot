# Подключение к удаленному серверу и отправка сообщений

Данный код реализует GUI чат клиента для подключения к удаленному серверу minechat.dvmn.org

## Как установить

Python3 уже должен быть установлен. Рекомендуется использовать venv для изоляции проекта.

Затем используйте pip для установки зависимостей:

```
    pip install -r requirements.txt
```

## Цель проекта

В рамках изучения asyncio (потоковый ввод/вывод) 

## Как работает

проект содержит несколько компонентов:
```

- reg_gui: UI для регистрации пользователя

- gui: UI основного клиента

- main.py: основной файл программы- файлы с конфигурацией в папке configs.

- файл конфигурации логгера

```

Скрипт main.py  несколько опциональных параметров cli. Параметры могут быть переданы через cli, env или указаны в конфигурационном файле. Приоритет параметров выгдлядит следующим образос cli->env->config.

## Пример запуcка из командной строки

### Запуск без указания параметроd (параметр text должен быть указан в env или в конфиге):
```

python main.py

```

### Запуск с параметрами
```

python main.pypy --host=some.host --port=10000 --token=some.token --usermane=some.username --text=some.text

```

## Описание аргументов


 - host - имя хоста для соединения

 - port - номер порта для соединения

 - token - используется при аутентификации


## Пример заполнения конфигурационнного файла

```
host=minechat.dvmn.org
port=5050
token=819a9078-92d8-11f0-a5a4-0242ac110003

```

reader.ini используется для чтения сообщений

sender.ini используется для аутентификации и отправки сообщений
